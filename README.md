# 🛰️ STM32 Embedded Systems & Lab Projects

This repository contains a collection of laboratory assignments and firmware development projects focused on **STM32 microcontrollers** using the **HAL (Hardware Abstraction Layer) library**.

## 🛠️ Tech Stack
*   **Microcontrollers:** STM32 Series (F4, F1, H7).
*   **IDE:** STM32CubeIDE.
*   **Language:** C (Embedded).
*   **Peripherals:** ADC, PWM, UART, EXTI, Timers.

---

## 📂 Project Highlights

*   **🐍 USART GLCD Snake Game:** A classic Snake game implementation using USART communication and a Graphical LCD.
*   **⏱️ Timer Interrupts:** Precise time-based operations and interrupt-driven firmware architecture.
*   **🕹️ PWM Control:** High-resolution signal generation for motor or LED brightness control.
*   **📡 ADC & EXTI:** Analog-to-digital data processing and external interrupt handling for real-time responsiveness.

---

## 🔧 Skills Demonstrated
*   Hardware-Software integration and debugging.
*   Low-level peripheral driver configuration.
*   Real-time system design and interrupt management.
